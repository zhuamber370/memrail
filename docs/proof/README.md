# Proof Pack (P0)

This folder contains small, verifiable artifacts you can link in announcements and docs.

## 1) Dry-run diff (proposal)
- File: `docs/proof/dry-run-response.json`
- Shows: `change_set_id`, summary, and diff ledger for a proposed write.

## 2) Commit (human approval)
- File: `docs/proof/commit-response.json`
- Shows: commit id + timestamp for an approved change.

## 3) Audit trail
- File: `docs/proof/audit-events-proof.json`
- Shows: audit events generated by tool=`proof`.

## 4) Incident / rejection example
- File: `docs/proof/incident-human-reject.md`
- Shows: how to prevent accidental/unsafe writes by reviewing diffs first.
